## 6. Interactivity, Hover Effects & Animations (apply throughout)

- Smooth scroll for all anchor navigation
- Navbar: background/shadow transition on scroll
- Buttons: color shift + slight scale-up on hover, ripple or underline animation on links
- Cards (products, features, reviews): lift-and-shadow on hover, image zoom-in on hover
- Section reveal animations on scroll (fade-in-up) using a lightweight library (e.g., AOS.js) or CSS `IntersectionObserver`
- Counters animate from 0 to target number when scrolled into view
- FAQ accordion: smooth height transition with icon rotation
- Sticky WhatsApp floating action button (bottom-right corner, pulsing glow animation) that links directly to WhatsApp chat at all times

## 7. Responsiveness

- Must be **fully responsive** across all breakpoints: mobile (≤480px), large mobile (≤767px), tablet (768–1024px), laptop (1025–1440px), large desktop (1441px+)
- Use fluid grid/flexbox layouts, relative units (%, rem, vw/vh), and responsive images (`srcset` or CSS `object-fit: cover`)
- Navigation collapses into a hamburger menu below tablet width
- Test touch-friendly tap target sizes (min 44px) for buttons/links on mobile

## 8. Technical Recommendations

- **Stack:** Plain HTML5, CSS3, vanilla JavaScript (or React if a more dynamic/component-based build is preferred) — keep it lightweight since this is a marketing/brochure site
- **Fonts:** Load via Google Fonts (Inter or Poppins)
- **Icons:** Font Awesome or Lucide icons
- **Map:** Google Maps embed iframe (no API key required for basic embed)
- **Reviews backend:** Google Sheets + Google Apps Script Web App (as detailed above)
- **Form-to-WhatsApp:** Pure JavaScript, no backend required
- **Performance:** Lazy-load images below the fold, minify CSS/JS, compress all images before deployment
- **SEO basics:** Proper meta title/description, semantic HTML tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`), alt text on all images, Open Graph tags for social sharing
- **Accessibility:** Sufficient color contrast between red/blue and backgrounds, keyboard-navigable menus and form, ARIA labels on interactive elements
